\mbox{\hyperlink{class_csound_performance_thread_a9d9c8ba5a14a8beed06b5e0a34e005a0}{Csound\+Performance\+Thread(\+Csound $\ast$)}} \mbox{\hyperlink{class_csound_performance_thread}{Csound\+Performance\+Thread(\+C\+S\+O\+U\+N\+D $\ast$)}}

Performs a score in a separate thread until the end of score is reached, the playback (which is paused by default) is stopped by calling \mbox{\hyperlink{class_csound_performance_thread_a934fb76c73af13cbfe675db7199c0b93}{Csound\+Performance\+Thread\+::\+Stop()}}, or an error occurs. The constructor takes a Csound instance pointer as argument; it assumes that csound\+Compile() was called successfully before creating the performance thread. Once the playback is stopped for one of the above mentioned reasons, the performance thread calls csound\+Cleanup() and returns.
